## Repository Description

This repository contains the implementation of **Deep Learning laboratory assignments** for the course **UCS761 – Deep Learning**.
The assignments demonstrate fundamental neural network concepts implemented using **Python, NumPy, and Matplotlib**.

---

## Assignments

### 1. Activation Functions

File: `102303439_Activation_Functions.py`

Implementation and visualization of commonly used activation functions such as:

* Sigmoid
* Tanh
* ReLU
* Leaky ReLU
* ELU
* Softmax
* Softplus
* Binary Step

Graphs are plotted to analyze the behavior of each activation function.

---

### 2. Perceptron Learning

File: `102303439_Perceptron_Learning.py`

Implementation of **MP neuron and Perceptron learning algorithm**.
The perceptron is trained to learn logical gates such as:

* AND
* OR
* NOR
* NOT

---

### 3. Backpropagation Algorithm

File: `102303439_Backpropagation.py`

Implementation of a **feedforward neural network with backpropagation**.
The model is trained on the **XOR dataset** to demonstrate learning using gradient descent.

---

### 4. Binary Classification

File: `102303439_Binary_Classification.py`

Binary classification implemented using a **logistic neuron with gradient descent**.
The model is trained on a dataset and visualizes decision boundaries and loss curves.

---

### 5. Handwritten Digit Classification

File: `102303439_Digit_Classification.ipynb`

Implementation of a neural network for **handwritten digit recognition**.
The model learns to classify digits using forward propagation and backpropagation.

---

### 6. Analysis of Digit Classification

File: `102303439_Digit_Classification_Analysis.ipynb`

Analysis of the digit classification model using **evaluation metrics and visualization techniques** to study training behavior and performance.
